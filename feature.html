Features

DESIGN

More than 20 unique pages

Each page is provided with at least two display options, feel free to experiment.

Color themes

Theme includes 4 color schemes and manual tuning. Change the template to your taste.

Custom template home page

Each block on the main page has 2 design. You can choose any one you like.

TEMPLATE

Bootstrap 3

Template provides all the advantages of this framework.

High working speed

PdoTools used for high-speed operation. Each implemented snippet uses a caching system MODX.

SEO optimized

Smart setting url, logic output title and description, and many other small details that affect the SEO.

ADMINISTRATION

Full multilingual support

All texts are inserted through the template lexicon. You can easily transfer the pattern to the desired language, without affecting the code.

3 languages included

We have included 3 languages: English, Ukrainian and Russian.

Special user to edit content

If you do not edit your own content and your assistant does not need to see all the features of MODX, give him a special only access the content.

Unisender

We have integrated a template subscription for one of the best, convenient and cheap email mailing services - unisender.

DEVELOPMENT

Developers friendly

We are developers and therefore we understand the needs of our colleagues for comfortable operation.

SASS included

We have included in the package all Less scripts, that you can change, all you want and how you want.

Extendable JS

All template js implemented using modules. You can easily add, modify, disable modules without worrying that something will break in the other functional.

Best practices in the development of corporate websites

We have development experience of corporate websites, and we decided to collect and transfer the best practices in the template.

Deploy the demo content

You can deploy the exact same demo site without any effort.
